### Requirements

Follow instructions [here](https://deno.land/#installation) to install Deno if

You will find your api key under the keys tab in your account settings

### Usage

Start the project:

```
MORALIS_WEB3_API_KEY=<YOUR_API_KEY> deno task start
```
